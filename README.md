# IVAO Copilot

A real-time flight monitoring service for IVAO pilots. Get Telegram notifications when ATCs come online along your flight route.

## Features

- 📍 Real-time flight monitoring
- 🔔 Telegram notifications when ATCs come online
- 📋 Easy setup with SimBrief flight plan
- 🛫 Track your route on IVAO

## Quick Start

1. Enter your IVAO VID (pilot ID)
2. Link your SimBrief flight plan
3. Connect your Telegram account
4. Get notified when ATCs appear on your route!

## Tech Stack

- **Frontend**: React + TypeScript
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Real-time**: Socket.io
- **Notifications**: Telegram Bot API
