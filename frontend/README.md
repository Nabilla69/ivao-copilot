# IVAO Copilot Frontend

React-based website for IVAO flight monitoring.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm start
```

The app will open at `http://localhost:3000`

## Build

```bash
npm run build
```

## Features

- Simple form to enter VID, Telegram ID, and SimBrief link
- Real-time feedback
- Responsive design
- Beautiful gradient UI
